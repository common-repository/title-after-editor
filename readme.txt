If you'd rather write your post title AFTER you write your post, this plugin moves the title field under the editor box.

== Description ==

This is a super-simple plugin. It does one thing: move the title field under the editor box when you're writing a post.

== Installation ==

1. Upload the `title-after-editor` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

or

1. Search for "Title After Editor" in the "Plugins" > "Add New" screen and install from there :)
